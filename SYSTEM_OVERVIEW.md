# 글로벌 야간 시황 숏츠 — 시스템 설명서

사람이 손 안 대도 **매일 여러 번 유튜브 쇼츠를 기획·제작·업로드**하는 AI 콘텐츠 시스템입니다.
GitHub Actions(클라우드)에서 돌기 때문에 PC를 꺼놔도 됩니다.

## 하루 자동 발행 (한국시간, 5~20분 늦을 수 있음)
- 06:17 아침 브리핑(morning) — 밤사이 미국장 + 오늘 국내 대응
- 08:17 개장 직전(preopen)
- 18:17 마감 복기(close)
- 22:17 미국장 프리뷰(uspreview)
- 주말=이번 주 TOP5 / 월=주간 전략 / 금=다음 주 준비 (planner가 자동 판단)

## AI가 도는 순서 (각 단계 = 하나의 역할)
1. collect — 지수·환율·금리·선물·코인 시세 + 뉴스 헤드라인 수집(무료)
2. researcher — 오늘 호재/악재 뉴스 + 향후 2주 일정(실적·CPI·금통위·IPO·만기) 조사
3. planner — 오늘 '조회수 가장 잘 나올 앵글' 1개 선택. 최근과 겹치면 전환
4. analyze — 그 앵글로 전략 브리핑 + 14개 장면 스토리보드 + 테마 작성
5. editor(편집장) — 발행 전 자기 채점. 미달이면 planner부터 재생성
6. fact_checker — 확인 안 되는 수급 금액 등 날조 제거
7. copywriter — 훅·제목 10개씩 만들어 CTR 예측 후 베스트 채택
8. images — 장면마다 내용에 맞는 AI 이미지(매번 다른 seed)
9. render — 한 화면 한 메시지(큰 글씨) + 테마별 색(반도체=초록/전력=파랑/비트=금색)
10. tts — 무료 뉴럴 음성(빠른 아나운서 톤)
11. make_video — 장면 길이 글자수 자동 배분 + 줌 교차 + 훅 펀치 + 크로스페이드
12. seo — 제목/설명/태그(호재·일정 캘린더 포함)/고정댓글
13. upload — 유튜브 무인 업로드
14. notify — 카카오톡 '나에게 보내기' 알림(설정 시)

## 필요한 GitHub Secrets
- GEMINI_API_KEY — AI 분석(필수)
- YT_CLIENT_ID, YT_CLIENT_SECRET, YT_REFRESH_TOKEN — 유튜브 업로드(필수)
- KAKAO_REST_KEY, KAKAO_REFRESH_TOKEN — 카카오 알림(선택, auth/get_kakao_token.py로 발급)

## 조정은 config.yaml 만
공개범위(public/unlisted/private), 음성/속도, 색, 수집 종목·뉴스, 알림 on/off

## 아직 안 된 것(다음 후보)
- 진짜 모션그래픽(그래프/GPU 애니메이션) — 프레임 캡처라 렌더 무거움
- 자막 CapCut식 단어별 팝업(현재는 장면 글자가 그 역할)
- 실제 조회수·CTR 학습 — 유튜브 Analytics 연동 + 권한 추가 필요

## 주의
- 종목·일정은 정보 제공용이며 투자 권유 아님(영상 하단 문구 자동).
- 일정 날짜는 AI 조사 기반 '예정' 개념으로 참고.
